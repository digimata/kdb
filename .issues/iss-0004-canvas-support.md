# ISS-0004 :: Canvas Support

Add support for `.canvas` artifacts so visual maps can participate in kdb indexing and navigation workflows.

- `.canvas` files are discoverable in the vault model.
- Canvas nodes can reference markdown notes and resolve reliably.
- CLI/LSP behavior is defined for canvas links and diagnostics.
